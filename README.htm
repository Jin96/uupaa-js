uupaa.js

uupaa.jsは、日本生まれのコンパクトでクロスブラウザなJavaScriptライブラリです。

“ライブラリの選択肢が増えるのは悪くない”と思える方は、一度デモをご覧になってください。

Feature comparison - 機能比較...

DEMO - デモ

簡単なデモを用意しました。これ以外にも動作確認用のhtmlが demo/ にあります(作りかけのものも混じっています)。

  • Effect: 基本エフェクト
  • UI: Drag & Drop - ドラッグ & ドロップ
  • 3D: Cube - キューブ, Cube x 8 - キューブ8個
  • Timer: Virtual Timer - 仮想タイマー
  • Request: Ajax, JSONP

アピールポイント

ウィークポイント

  1. MITライセンスのオープンソースです。
  2. 豊富な日本語ドキュメント。これからの人にやさしい。
  3. モジュールのオンデマンドロード, MVC, singleton等々、
    クリティカルな機能を提供します。
  4. Cover Flowやポリゴン, Text-To-Speechなど、
    他には無い特別な機能があります。
  5. IEでもしっかりcanvasが使えます。
  6. セレクタが豊富です。
    XPath, CSS3セレクタ, jQueryの独自セレクタも使用可能です。
  7. カジュアルで軽量なUIが(近い将来)使えます。
  1. 実績がありません。
  2. 一部のモジュールのドキュメントがまだ存在しません。

Support Browser - サポートブラウザ

  • Internet Explorer 6+ (IE5.5でも動くかもしれないが未検証のためサポート外)
  • Safari3+ (Safari2はサポート外)
  • Opera9+ (canvasやXPathなどが使用できないためOpera8以下はサポート外)
  • Firefox2+ (Firefox1.5系以下は未検証のためサポート外)

Opera9.5, Firefox3, Internet Explorer 8β1 でも動作を確認しています。(IE8βには未実装の機能があり、その影響でいくつかのデモは動きません)

Installation - 導入方法

uupaa.jsを使用したページをIEで動かすためには、 JavaScript-XPathと ExplorerCanvasが必要になります。

JavaScript-XPathは /lib/javascript-xpath.js として uupaa.js に同梱されていますが、 ExplorerCanvasは非同梱です。配布元からダウンロードし、例を参考に配置してください。

Folder tree - ファイルの配置例

http://www.example.com/js/ 以下にuupaa.jsを設置する例です。

  • http://www.example.com/
    • js/
      • uupaa.js
      • uu.module.drag.js - (ドラッグ&ドロップモジュール)
      • uu.module.xxx.js ...
      • demo/ - (デモ/テスト用ディレクトリ)
      • img/ - (画像,CSS用ディレクトリ)
      • lib/ - (外部ライブラリ用ディレクトリ)
        • javascript-xpath.js - (JavaScript-XPath) - IEで必要
        • excanvas.js - (ExplorerCanvas) - IEで必要
      • mini/ - (圧縮済みファイル用ディレクトリ)
        • uupaa.js - (圧縮済みのuupaa.js)
        • uu.module.xxx.js ...
