uupaa.js

uupaa.jsは、日本人クリエーター向けのJavaScriptライブラリです。クロスブラウザ対応です。

uupaa.jsは生まれたばかりで、まだ実績がありません。
jQuery や Prototype.js ユーザからみれば「あれも無い。これも無い」といった状態です。
“選択肢が増えるのは悪くない”と思える方は、一度デモをご覧になってください。

アピールポイント

ウィークポイント

  1. MITライセンスのオープンソースです。
  2. 豊富な日本語ドキュメント。これからの人にやさしい。
  3. モジュールのオンデマンドロード, MVC, singleton等々、
    クリティカルな機能を提供します。
  4. Cover Flowやポリゴン, Text-To-Speechなど、
    他には無い特別な機能があります。
  5. canvasによるフロントエンドにフォーカスしています(いきます)。
  6. セレクタが豊富です。
    XPath, CSS3セレクタ, jQueryの独自セレクタも使用可能です。
  1. 実績がありません。
  2. とても流動的な状態です。
    メソッド名の変更など、痛い修正が十分にありえます。
  3. ドキュメント周りがまだまだです。
  4. デバッグ支援用の機能がまだまだです。
  5. from周りのサポートが全く実装されていません。
  6. プログレスバーやスライダー等のウィジェットが全く実装されていません。
    jQueryの機能をエミュレートするuu.module.jqssにより、
    jQueryのプラグインが一部動作するようになりました。

サポートブラウザ

  • Internet Explorer 6+ (IE5.5でも動くかもしれないが未検証のためサポート外)
  • Safari3+ (Safari2のバグに対処できていないためサポート外)
  • Opera9+ (canvasやXPathなどが使用できないためOpera8以下はサポート外)
  • Firefox2+ (Firefox1.5系以下は未検証のためサポート外)

Opera9.5β2, Firefox3β5, Internet Explorer 8β1 でも動作を確認しています。(IE8βには未実装の機能があり、その影響でいくつかのデモは動きません)

導入方法

uupaa.jsを使用したページをIEで動かすためには、 JavaScript-XPathと ExplorerCanvasが必要になります。

これらは非同梱です。配布元からダウンロードし、例を参考に配置してください。

<style type="text/css">
#canvas, #msg { position: absolute; top: 0; left: 0 }
#msg { font-weight: bold }
</style>
</head>
<body>
<a href="../../README.htm">README.htm</a> &gt; HelloWorld.htm

<canvas id="canvas" width="400" height="400"></canvas><p id="msg">Hello Japanese Creator!</p>
<script type="text/javascript">
function circle() { 
  var canvas = uu.id("canvas"), ctx = canvas.getContext("2d"), color = 0xff9999;
  function draw() {
    color += 10;
    ctx.clearRect(0, 0, ctx.canvas.width, ctx.canvas.height);
    ctx.beginPath();
    ctx.fillStyle = color.toRGBString();
    ctx.arc(200, 200, 180, 0, Math.PI * 2, false);
    ctx.fill();
  }
  uu.effect.move(uu.id("msg"));
  uu.effect.move(canvas);
  uu.tm10.set(function() {
    uu.effect.fade(canvas, { speed: 2000, fn: draw })
  }, 2200);
}
uu.module.load("", "ui,effect", function() {
  uu.dom.ready(function() {
    uu.canvas.ready(circle);
  });
});
</script>
</body>
</html>

デモ

簡単なデモを用意しました。これ以外にも動作確認用のhtmlが demo/compat/ にあります(作りかけのものも混じっています)。

  • Ajax, JSONP
  • window.ready, dom.ready, canvas.ready, image.preload(画像の先読み)
  • エフェクト - Version 0.2で11個のエフェクトを追加しました
  • 透過PNG(IE5.5/IE6.0専用)
  • ドラッグ & ドロップ
  • キューブ, キューブ8個
  • 仮想タイマー
  • Cover Flow(作りかけ)(Firefox2のみ)
  • ペリーさん(作りかけ)(IE or Firefox + IETabユーザのみ)
