uupaa.jsは、日本人クリエーター向けのJavaScriptライブラリです。クロスブラウザ対応です。

uupaa.jsは生まれたばかりで、まだ実績がありません。
jQuery や Prototype.js ユーザからみれば「あれも無い。これも無い」といった状態です。
“選択肢が増えるのは悪くない”と思える方は、一度デモをご覧になってください。

アピールポイント

ウィークポイント

  1. MITライセンスのオープンソースです。
  2. 日本語のドキュメントが豊富。
    これからの人にやさしい。
  3. モジュールのデマンドロードに対応しています。
  4. カバーフローやポリゴン, Text-To-Speechなど、
    他には無い特別な機能があります。
  5. canvasによるフロントエンドにフォーカスしています(いきます)。
  6. 大規模開発も考慮し、プラガブルMVCやシングルトンクラスなどが利用可能。
    メッセージポンプで楽しましょう。
  1. 実績がありません。
  2. とても流動的な状態です。
    メソッド名の変更など、痛い修正が十分にありえます。
  3. ドキュメント周りがまだまだです。
  4. デバッグ支援用の機能がまだまだです。
  5. from周りのサポートが全く実装されていません。
  6. プログレスバーやスライダー等のウィジェットが全く実装されていません。

サポートブラウザ

  • Internet Explorer 6+ (IE5.5でも動くかもしれないが未検証のためサポート外)
  • Safari3+ (Macで動かしたことがないのと、Safari2のバグにどう対処すればよいのかが良くわからないのでサポート外)
  • Opera9+ (canvasやXPathなどが使用できないためOpera8以下はサポート外)
  • Firefox2+ (Firefox1.5系以下は未検証のためサポート外)

Opera9.5β2, Firefox3β5, Internet Explorer 8β1 でも動作を確認しています。(IE8βには未実装の機能があり、その影響でいくつかのデモは動きません)

導入方法

IEでuupaa.jsを使用したサイトを動かすためには、 JavaScript-XPathと ExplorerCanvasが必要になります。

これらは同梱していませんので、配布元からダウンロードを行い、以下の例を参考に配置してください。

http://www.example.com/js/ 以下にuupaa.jsを設置する例です。

シンプルな例 - HelloWorld.htm

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
  "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="ja">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
<title>Hello uupaa.js World</title>
<!--[if IE]><script id="xpath.js" type="text/javascript" src="../../lib/xpath.js"></script><![endif]-->
<!--[if IE]><script id="excanvas.js" type="text/javascript" src="../../lib/excanvas.js"></script><![endif]-->
<script id="uupaa.js" type="text/javascript" src="../../uupaa.js"></script>
<style>
#canvas, #msg { position: absolute; top: 0; left: 0 }
#msg { font-weight: bold }
</style>
</head>
<body>
<canvas id="canvas" width="400" height="400"></canvas><p id="msg">Hello Japanese Creator!</p>
<script>
function circle() { 
  var canvas = uu.id("canvas"), msg = uu.id("msg"),
      ctx = canvas.getContext("2d"),
      inn = uu.ui.inner(),
      color = 0xff9999;
  function alignCenter() {
    [msg, canvas].forEach(function(v) {
      uu.css.set(v, { left: (inn.iw - uu.css.get.width(v)) / 2 + "px",
                      top: (inn.ih - uu.css.get.height(v)) / 2 + "px" });
    });
  }
  function draw() {
    color += 10;
    ctx.clearRect(0, 0, ctx.canvas.width, ctx.canvas.height);
    ctx.beginPath();
    ctx.fillStyle = color.toRGBString();
    ctx.arc(200, 200, 180, 0, Math.PI * 2, false);
    ctx.fill();
  }
  function anim() {
    uu.effect.fade(canvas, { speed: 2000, fn: draw });
  }
  alignCenter();
  uu.tm10.set(anim, 2200);
}
uu.module.load("../../", "ui,effect", function() {
  uu.dom.ready(function() {
    uu.canvas.ready(circle);
  });
});
</script>
</body>
</html>
IEでは、xpath.jsやexcanvas.jsをuupaa.jsの前に読み込んでください。読込に失敗するとエラーが発生したりcanvasが使用不能になります。 他のブラウザではこれらのライブラリは不要です。

デモ

簡単なデモを用意しました。これ以外にも動作確認用のhtmlが demo/compat/ にあります(作りかけのものも混じっています)。

  • Ajax, JSONP
  • window.ready, dom.ready, canvas.ready, image.preload(画像の先読み)
  • エフェクト(Fadeのみ)
  • 透過PNG(IE5.5/IE6.0専用)
  • ドラッグ & ドロップ
  • キューブ, キューブ8個
  • 仮想タイマー
  • Cover Flow(作りかけ)(Firefox2のみ)
  • ペリーさん(作りかけ)(IE or Firefox + IETabユーザのみ)
