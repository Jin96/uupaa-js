Class uu.module.virtualTimer


Defined in: uupaa.js.

Class Summary
Constructor Attributes Constructor Name and Description
 
uu.module.virtualTimer()
Quick Timer ベースとなるタイマーで時刻を刻み、 その上で複数の仮想タイマーを動作させることで、コンテキストスイッチを減らし、 100個以上のタイマーを高速に動作させることができる高機能タイマーです。
一時停止、再開、ループ回数の指定、遅延時間の動的な指定が可能です。
タイマーの精度は、setIntervalやsetTimeoutよりは落ちます(そもそもそれらも正確ではありません)。
タイマー周期は約27万年分ほどあるのでオーバーフローの心配はありません。
