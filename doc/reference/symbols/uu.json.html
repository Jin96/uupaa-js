<static> uu.json.load(string, function)
uu.json.load - 非同期ロード[JSONP async request] JSONPで非同期にデータを取得します。 自動的にコールバック関数(uu.json._cache._{uid})が定義され、この関数名でラップされたJSONデータがサーバから返却されます。
ロード完了から60秒後にリクエストに使用したリソースを開放します。

step番号内容
0BEGIN: 処理開始
1SEND: リクエスト開始
2OK: 成功
3NG: 失敗
4END: 処理終了
各ステップで、fn(ユニークID, step番号, response, status, url, async)を呼び出します。
responseには空文字列またはレスポンス文字列が格納されます(OKのみ)。
statusには0またはステータスコードが格納されます(OKとNGのみ)。
asyncは常にtrueになります。
Parameters:
string
url - リクエストURLを指定します。
function
[fn] - 各stepで呼び出す関数を指定します。デフォルトはundefinedです。
