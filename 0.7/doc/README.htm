uupaa.js

高速でコンパクト, 未来指向の JavaScript ライブラリ

Ready

  • IE 6+
  • Firefox 3+
  • Opera 9.5+
  • Chrome
  • Safari 3.1+
  • iPhone

Features

  • HTML5, CSS3
  • uuAltCSS, uuCanvas, uuQuery
  • Ajax, Color, ConditionalSelector, ECMAScript5th, Event, FlashLoader, JSON, jQuery Plugins, Layer, MessagePump, NodeBuilder, Tween, UI, UnitTest, WebStorage

Details

  • HTML5 APIs
    <canvas> → uu.canvas()
    WebStorage → uu.local()
  • CSS3 APIs
    document.querySelectorAll → uu.query()
    document.matchesSelector → uu.match(), uu.live()
  • CSS2.1 and CSS3
    CSS2 MaxWidth → max-width: (for IE)
    CSS3 Opacity → opacity: (for IE)
    CSS3 Display:table → -uu-display: table
    CSS3 Gradients → -uu-gradient(...)
    CSS3 BorderRadius → -uu-border-radius:
    CSS3 DropShadows → -uu-box-shadow: -uu-text-shadow:
    CSS3 MultipleBackgrounds → -uu-background:
    CSS3 Reflection → -uu-box-reflect:
    WebKit BackgroundCanvas → -uu-canvas()
  • CrossBrowser APIs
    Ajax → uu.ajax()
    CanvasRenderingContext
    Color → uu.color()
    DOM → uu.node(), uu.nodeid()
    ConditionalSelector
    Cookie → uu.cookie()
    Drag&Drop → uu.Class.Drag()
    Event → uu.ev()
    FlashLoader → uu.flash()
    Form → uu.text(), uu.val()
    Image → uu.img()
    JSON → uu.str2json(), uu.mix2json(), uu.json2mix()
    JSONP → uu.jsonp()
    Tween → uu.tween()
    Type detection → uu.type(), uu.like(), uu.has(), uu.isxxx()
    Version detection → uuvers()
    window.getComputedStyle (for IE)
