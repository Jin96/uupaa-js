以下の制限があります。

uupaa.js では、-uu-background-image, -uu-background-repeat, -uu-background-position により、 CSS3 の multiple background image に近い表現が可能になります。

-uu-background-repeat は "no-repeat" のみサポートします(ソフトでタイリングを行うとレスポンスが心配なので)

-uu-background-image は、-uu-gradient() をサポートしません。

-uu-background-image と同じ数の、-uu-background-repeat と -uu-background-position を指定する必要があります。

IEでは、背景画像をボーダーの内径に収まるようにクリッピングしていません(uu.canvas.js の VMLモードの制限)。そのため、角丸の上に背景画像が表示されるケースがあります。角丸と同時に使う場合は、Silverlightでレンダリングするか、角の近くに背景画像を設置しない工夫が必要です。

Safari4+, Chrome2+ では、-uu-background-image, repeat, position は利用できません。 それらには、multiple background image 本来の書式で背景を指定します。
ブラウザの切り分けには、コンディショナルセレクタ( html.if{xxx} )を利用します。

html.ifnojs .mbg1 {
  /* JavaScript OFF の環境用 */
  background: url(solareclipse.jpg) repeat center centet;
}
html.ifjs .mbg1 {
  /* Safari4+, Chrome2+ 以外の環境用(uupaa.js の機能で複数の背景画像を表示する) */
  background: url(solareclipse.jpg) repeat center center;
  -uu-background-image:
      url(../alphapng/c32.alpha.png), url(../alphapng/f32.alpha.png),
      url(../alphapng/e32.alpha.png), url(../alphapng/o32.alpha.png),
      url(../alphapng/s32.alpha.png),
      url(stage.png);
  -uu-background-repeat:
      no-repeat, no-repeat, no-repeat, no-repeat, no-repeat,
      no-repeat;
  -uu-background-position:
      33% 98%, 42% 98%, 50% 98%, 58% 98%, 67% 98%,
      center bottom;
}
html.ifwebkit .mbg1 {
  /* Safari4+, Chrome2+ 用(CSS3 本来の書式で複数の背景画像を表示する) */
  background-image:
      url(../alphapng/c32.alpha.png), url(../alphapng/f32.alpha.png),
      url(../alphapng/e32.alpha.png), url(../alphapng/o32.alpha.png),
      url(../alphapng/s32.alpha.png),
      url(stage.png),
      url(solareclipse.jpg);
  background-repeat:
      no-repeat, no-repeat, no-repeat, no-repeat, no-repeat,
      no-repeat,
      repeat;
  background-position:
      30% 98%, 40% 98%, 50% 98%, 60% 98%, 70% 98%,
      center bottom,
      center center;
}
