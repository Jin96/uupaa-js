Array.indexOf - 配列の先頭から値を検索し最初のindexを返す。無ければ-1を返す(JavaScript 1.6準拠)

rv["hash"] = "hash";
rv.indexOf(1), rv.indexOf(1, -2); // 1, 2
rv.indexOf("1"), rv.indexOf("hash"); // -1, -1
