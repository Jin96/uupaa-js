Function Namespace uu.css.set

uu.css.set - スタイルの設定 - set style スタイルを設定します。
一部のプロパティでは、特別な値や指定方法が有効です。

top, left, width, height
値を数値で指定すると末尾に"px"を補完します。
display
"hide"を指定すると要素を隠します。"show"を指定すると要素を表示します。
opacity
不透明度を0.0から1.0の値で指定可能

Defined in: uupaa.js.
